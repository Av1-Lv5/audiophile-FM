# Audiophile | E-Commerce

A multi-page e-commerce site for an audio electronics selling business, built with Next.js 14 and Sanity CMS.

## Tech stack

-   Sanity CMS
-   Next JS
-   Tailwind CSS
-   React
-   TypeScript

The designs are from a [frontend mentor challenge](https://www.frontendmentor.io/challenges/audiophile-ecommerce-website-C8cuSd_wx)

> Frontendmentor is a platform offering, some free and paywalled professional designs for frontend developers to create stunning projects.

View my list of all solved Frontementor challenges [here](https://github.com/Av1-Lv5/Frontendmentor-challenges)

## Clone this?

Follow these steps if you want to clone this and play around with the code.

-   `pnpm i` if you want to use some other package manager like yarn or npm, delete the pnpm-lock.yaml file before doing so.
-   Start tinkering.
-   you need to have a sanity account.
-   you can find the env variables required in `.env.example`.
-   update the sanity studio with data, to see the website in action.
